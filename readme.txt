A simple widget which shows thumbnails of latest photos in posts

== Description ==

A plugin offering a widget showing recent photos in posts. It can be configured
as for how many photos to show, the title on the widget and max width and height for
thumbnails. 

The thumbnails links to the post where the photo is showing. 

It has only been tested with WP 4.2, it might work with lower versions. 


== Installation ==

1. Upload `recent_photos_widget.zip in wp-admin and install it
2. Activate the plugin 
3. Add the widget to a sidebar
